Overrides

This folder contains resource name overrides in the form:
Old Name|New Name

example:
_rsv_30478_-1_1_0_0_SE2DC5B04_EE2DC5B04|Claw to Tail

It is preferable to group them into separate files so they can be maintained independently,
perhaps by zone or any other sane delimiter.  The file name does not matter.
